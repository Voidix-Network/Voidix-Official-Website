# Voidix官方网站

Voidix是一个专业的Minecraft小游戏服务器，为玩家提供免费、公平的休闲游戏体验。

## 🌟 特色功能

- 🎮 多样化的Minecraft小游戏
- 🔗 支持跨版本连接（Java & Bedrock）
- 🆓 完全免费，无付费内容
- ⚖️ 公平竞技环境
- 📱 响应式网站设计
