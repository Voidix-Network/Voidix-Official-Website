Bug 反馈

我们非常重视您在游戏过程中遇到的任何问题。如果您发现了Bug，请尽可能详细地告诉我们，这将帮助我们更快地定位和修复问题。

反馈渠道：

  • 通过官方QQ群： 186438621 (推荐，可截图和实时交流)
  • 通过Discord服务器： 加入我们的 Discord
  • 通过邮件发送至： support@voidix.net
  • 通过Github Issues： 本网站 GitHub Issues 页面

反馈时请尽量包含以下信息：

  • 您当时所在的服务器 (例如：小游戏服、生存服)
  • 您正在游玩的游戏模式或具体场景
  • Bug发生的具体时间和日期 (大致即可)
  • Bug的详细描述 ：您做了什么操作？预期结果是什么？实际发生了什么？
  • 复现步骤 ：如果可能，请提供如何让Bug再次出现的步骤。
  • 截图或录屏 ：这对于我们理解问题非常有帮助。
  • 您的游戏内ID

感谢您的帮助，让我们一起把Voidix建设得更好！
